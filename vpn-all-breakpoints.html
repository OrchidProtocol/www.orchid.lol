A BANDWIDTH MARKETPLACE

The Orchid VPN uses a stake-weighted algorithm to randomly connect users to one of Orchid's preferred bandwidth providers.

ORCHID ACCOUNTS

The foundation of the Orchid user experience. The Orchid account manages your funds and enables access to decentralized marketplaces.

PAY-PER-USE

Users never pay for service they don’t get. When you use Orchid’s VPN protocol, nanopayment tickets are sent alongside your data packets, delivered straight to your bandwidth provider.

PREFERRED PROVIDERS

Orchid doesn’t run servers on the VPN protocol; instead, independent providers run the Orchid server configuration and stake OXT to provide bandwidth in exchange for payments directly from users via our nanopayment system.

WHY ORCHID VPN?
OPEN SOURCE

All of Orchid's code is Open Source and freely available to download on GitHub. Use of Orchid's source code is governed by the AGPL copyleft Open Source license. The Orchid community develops code "in the open" by continually pushing changes that anyone can see on GitHub and tagging releases as appropriate.

CHECK OUT ORCHID’S GITHUB
NO SUBSCRIPTIONS

Through the power of our nanopayment ecosystem, you only pay for bandwidth when you’re using the VPN. No monthly subscriptions, no long-term contracts, just service as you need it.

TRACKING-FREE

Orchid has no tracking pixels, third-party JavaScript, or any code on our website that tracks your information, and that same mindset extends to our projects and applications.

DECENTRALIZED

Through Orchid’s decentralized bandwidth marketplace and stake-weighted randomized server selection, users are not beholden to trust a single ISP with all their traffic. This multi-player market and selection randomization makes user information more difficult for would-be trackers and hackers to exploit.

BYPASS RESTRICTIONS

Use Orchid’s multi-hop configuration to construct routes around firewalls or regionally-locked content.

WIREGUARD & OPENVPN INTEGRATION

The Orchid VPN supports WireGuard and OpenVPN connections in our multi-hop route configurations.
